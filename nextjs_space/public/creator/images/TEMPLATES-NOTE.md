# Creator template SVGs – rules

**Do not change the original look of templates.**

- Every template SVG must look exactly as designed. No visual alterations (no replacing path-drawn text with approximated curves, no hiding original art to show different text).
- When adding or supporting editable text, **keep the original form intact**. If a design uses curved or path-based text, make it editable only when the source file provides real `<text>` / `<textPath>` elements—do not modify the template to substitute a different look.
- Same for colors, logos, and layout: preserve the original; add editability in code (parser/UI) without changing how the template appears by default.
